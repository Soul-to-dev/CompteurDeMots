# Compteur de Mots - Java

## Description
Programme Java qui lit un fichier texte et compte
le nombre de mots dans chaque ligne ainsi que le total.

## Fonctionnalités
- Lecture d'un fichier texte ligne par ligne
- Comptage des mots par ligne
- Affichage du contenu de chaque ligne
- Affichage du nombre total de lignes et de mots

## Structure du projet
- `Main.java` : contient tout le code du programme
- `texte.txt` : fichier texte utilisé pour le test

## Comment exécuter
1. Cloner le projet : `git clone https://github.com/Soul-to-dev/CompteurDeMots.git`
2. Ouvrir dans IntelliJ IDEA
3. Lancer `Main.java`
4. Le programme lit automatiquement le fichier `texte.txt`
